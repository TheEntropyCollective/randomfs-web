About RandomFS

RandomFS is a modern implementation of the Owner Free File System (OFF System) concept, built on top of IPFS. It stores files as randomized, multi-use data blocks that appear as digital noise individually but can be combined to reconstruct the original files.

Key Features:

  • Files are split into randomized blocks stored on IPFS
  • Each block serves multiple files simultaneously
  • Access via rd:// URLs for decentralized retrieval
  • Three block sizes optimized for different file categories
  • Local caching for improved performance

rd:// URL Format: rd://host/version/filesize/filename/timestamp/hash
